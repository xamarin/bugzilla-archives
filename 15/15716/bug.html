Description Alfred Hall 2013-10-25 16:18:20 UTC 
Mono is crashing under heavy load. I got a very simple selfhosted NancyFX webservice that's inserting users into cassandra using the datastax csharp driver, I'm using the LINQ driver to make my life easier. In order to make testing easy I issue a get request against /account/registerandom and it does a couple of cassandra queries and creates a random username and password. Self hosted NancyFX uses System.Net.HttpListener and then passes the data received on to the Nancy pipeline.

Here is what I do to reproduce the issue:
1. Start the self hosted NancyFX like this:

MONO_THREADS_PER_CPU=100 /opt/ahall-mono/bin/mono-sgen TestNancy.exe

Then keep this running.

2. Start 4 instances on this in 4 different ssh sessions:
ab -c 50 -n 1000000 "http://127.0.0.1:8989/account/registerrandom2
