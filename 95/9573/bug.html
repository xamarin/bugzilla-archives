Description Clay Fowler 2013-01-15 11:35:14 UTC

Also similar hang when trying to build solution on previously created MonoMac projects. Can't say for sure that the problem has anything to do with MonoMac or with the update to 3.0.6, but never had any problems prior to the 3.0.6 update today. 

I've tried both with the recommended 2.10.9 MDK in MonoDevelop's updater, as well as the latest 3.0.2 MDK. Same result both ways.
