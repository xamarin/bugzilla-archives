Full Error Message when opening storyboard file

*Overview*
When attempting to open a pre-existing or new storyboard file in Xamarin Studio, the file fails to open with the below error:

**Note**: Full details from error in attachment

System.AggregateException: One or more errors occured ---> System.SystemException: RegisterServicePort(com.xamarin.MTHosting.4e67b7042, com.apple.PowerManagement.control): Kernel returned: -308 (-308): (ipc/mig) server died


*Steps to reproduce*
1) Create a new Xamarin.iOS project that includes a storyboard file.
2) Attempt to open the storyboard file.

*Expected Results*
Users should be able to open the storyboard file with no error message.

*Actual Results*
Users are unable to open the storyboard file and met with the previously mentioned (and attached) error message.

*Additional Information*
The error message has changed since the update to XS 5.2, although the "Kernel returned: -308" message seems consistent through the versions where this issue has occurred.

The storyboard file can be opened in Xcode with no error.

I also added the storyboard file and logs from XS to this report.
